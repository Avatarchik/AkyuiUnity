# AkyuiUnity

AkyuiUnity is a Unity implementation of [Akyui](https://github.com/kyubuns/Akyui).  
With AkyuiUnity.Xd, you can easily generate Unity Prefab from [Adobe XD](https://www.adobe.com/products/xd.html) files.
