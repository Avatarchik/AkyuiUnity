# AkyuiUnity

AkyuiUnity is a Unity implementation of [Akyui](https://github.com/kyubuns/Akyui).  
AkyuiUnity.Xd allows you to generate Unity Prefab from [Adobe XD](https://www.adobe.com/products/xd.html).
